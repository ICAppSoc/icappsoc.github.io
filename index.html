Android Session One - The Force Awakens

Please download and install the following tools, in order:

Java Development Kit 7+ - once installed, select Start menu > Computer > System Properties > Advanced System Properties. Then open Advanced tab > Environment Variables and add a new system variable JAVA_HOME that points to your JDK folder, for example C:\Program Files\Java\jdk1.7.0_40.

Android SDK Tools - under GET THE SDK FOR AN EXISTING IDE. Run the installer and keep note of the location where you install it.

Android Studio - this is now a portable application, i.e. you do not install it but just run it from /bin/studio(64).exe. When you run it for the first time, go to Configure > Project Defaults > Project Structure and enter the location of your Android SDK folder that you noted in the previous step. If you have a project open, you can change the SDK location under File > Project Structure.

To start developing, launch the Android SDK Manager and install:

Under Tools

  • Android SDK Tools
  • Android SDK Platform-tools
  • Android SDK Build-tools

Under Android 5.0

  • SDK Platform
  • Intel x86 Atom System Image
  • Google APIs

Under Extras

  • Android Support Repository
  • Android Support Library
  • Intel x86 Emulator Accelerator (HAXM)

To run applications, you have two choices:

Using physical Android device

Using virtual Android device (AVD)

  1. Go to %ANDROID_SDK%/extras/intel/HAXM and run the installer.
  2. Find and run the Android Virtual Device (AVD) Manager.
  3. Go through the wizard to create a virtual device, choosing the x86 image.

Then just click Run in Android Studio, and if the project is error-free, it will let you choose your hardware device or launch your emulator.
